Example: Mouse Pointer Position with Color

  • The v-on directive is used on the <div> element to listen to the 'mousemove' event.
  • When the 'mousemove' event occurs the 'mousePos' method is called
  • and the event object is sent with the method by default so we can get the mouse pointer position
  • if we expand the code by one line we can make the background color change based on the mouse position in the x-direction
  • The only thing we need to add is v-bind to change the background-color in the style attribute

Move the mouse pointer over the box below:

xPos: {{ xPos }}
yPos: {{ yPos }}

CSS:
backgroundColor:'hsl({{xPos}},80%,80%)'

To understand how to set a color in CSS with 'hsl()' see our page about this.
